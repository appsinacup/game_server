    GameServer.Theme.JSONConfig (GameServer v1.0.575)

    Copy Markdown

    JSON-backed Theme provider. Reads a JSON file specified by the THEME_CONFIG environment variable (single canonical runtime source) — e.g. THEME_CONFIG=theme/custom.json

    The path may be relative to the project root (eg. "theme/default_config.json") or an absolute path. When the file is missing we fall back to the built-in default at priv/static/theme/default_config.json.

    This implementation keeps things simple: every call will parse the JSON file and return a map. There's also a reload/0 API for callers who want to force a re-read (not required for normal usage).
