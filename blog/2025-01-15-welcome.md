# Welcome to Gamend

This is the first blog post for the Gamend game server project. We're excited to share updates about the development progress.

## What is Gamend?

Gamend is an open-source game server built with Elixir and Phoenix, providing features like:

- Authentication (Email, OAuth, JWT)
- Lobbies and matchmaking
- Friends and groups
- Leaderboards
- Server scripting
- Real-time updates via WebSockets
