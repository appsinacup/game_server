    Behaviour for object-storage backends.

    Implemented by GameServer.Storage.Local (disk, the dev default) and GameServer.Storage.S3 (any S3-compatible service — AWS S3, Cloudflare R2, Backblaze B2, MinIO, DigitalOcean Spaces). Callers go through the GameServer.Storage facade, never an adapter directly.

    A stored object's metadata, as listed by the admin tools.

    presigned()

    An upload ticket handed to a client so it can upload bytes directly to the backend (S3/R2) or to the local upload endpoint — the client flow is identical either way.
