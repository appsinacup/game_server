    GameServer.KV (GameServer v1.0.433)

    Generic key/value storage.

    This is intentionally minimal and un-opinionated.

    If you want namespacing, encode it in key (e.g. "polyglot_pirates:key1"). If you want per-user values, pass user_id: ... to get/2, put/4, and delete/2.

    This module uses the app cache (GameServer.Cache) as a best-effort read cache. Writes update the cache and deletes evict it.
