# GameServer v1.0.967 - API Reference

## Modules

- [GameServerHost.Router](GameServerHost.Router.md): Host-owned router for the running application.
- [GameServerHost.SitemapController](GameServerHost.SitemapController.md): Host-owned `sitemap.xml` controller.
- [GameServerWeb.HostBlogLive](GameServerWeb.HostBlogLive.md): Host-owned LiveView for the blog section.
- [GameServerWeb.HostChangelogLive](GameServerWeb.HostChangelogLive.md): Host-owned LiveView that renders the project changelog from Markdown content.

- [GameServerWeb.HostContentAssetController](GameServerWeb.HostContentAssetController.md): Serves static assets (images, etc.) from host-registered content sources.
- [GameServerWeb.HostPublicDocs](GameServerWeb.HostPublicDocs.md): Host-owned static LiveView that renders setup guides and product docs.

- [GameServerWeb.HostPublicDocsTemplates](GameServerWeb.HostPublicDocsTemplates.md): Embedded HEEx templates used by the host-owned public docs LiveView.

- [GameServerWeb.HostRoadmapLive](GameServerWeb.HostRoadmapLive.md): Host-owned LiveView that renders the project roadmap from Markdown content.
